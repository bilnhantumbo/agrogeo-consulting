# Brainstorming de Design - AgroGeo Consulting

## Contexto
Site institucional para consultoria técnica em agricultura, meio ambiente e recursos hídricos. Público-alvo: ONGs, projetos de desenvolvimento sustentável, instituições. Requisitos: cores verdes e azuis, layout limpo, profissional.

---

## <response>

### Abordagem 1: Modernismo Sustentável com Geometria Orgânica

**Design Movement:** Modernismo Sustentável + Biomimética Digital

**Core Principles:**
- Formas orgânicas inspiradas na natureza (curvas suaves, padrões naturais)
- Hierarquia clara através de espaçamento generoso e tipografia deliberada
- Integração visual de dados e geotecnologia como elemento central
- Transições fluidas entre seções que evocam fluxo de água e crescimento

**Color Philosophy:**
- Verde profundo (base): #2D5016 para confiança e expertise técnica
- Azul-água (acentos): #0E7C86 para inovação e recursos hídricos
- Neutro quente (fundo): #F8F9F6 para acessibilidade e leveza
- Raciocínio: Verde representa sustentabilidade e expertise florestal; azul representa água e tecnologia; neutro quente evita frieza institucional

**Layout Paradigm:**
- Seções com diagonal cuts e clip-paths suaves (não retos)
- Assimetria intencional: conteúdo à esquerda, imagens/dados à direita em alternância
- Grid de 3 colunas para serviços, mas com elementos que "transbordam" visualmente
- Hero section com fundo gradiente verde-azul com padrão de folhas sutilmente texturizado

**Signature Elements:**
1. Ícones customizados com linhas orgânicas (não geométricas)
2. Padrão de fundo: linhas de contorno topográfico sutil
3. Cards com sombra suave e borda verde-azul de 2px

**Interaction Philosophy:**
- Hover em cards: elevação suave + mudança de cor de borda para azul
- Links: underline animado em verde que expande ao hover
- Scroll: revelação gradual de elementos com fade-in

**Animation:**
- Entrada de seções: fade-in com slide suave de baixo para cima (200ms)
- Ícones em serviços: rotação suave ao hover (180°, 300ms)
- Números/métricas: contagem animada ao entrar em viewport
- Transição entre páginas: fade rápido (150ms) sem movimento

**Typography System:**
- Display: Playfair Display (bold, 48-56px) para títulos principais
- Heading: Lora (semibold, 28-32px) para subtítulos
- Body: Inter (regular, 16px) para conteúdo
- Rationale: Playfair adiciona elegância e autoridade; Lora é humanista e confiável; Inter é clara e moderna

### Abordagem 2: Minimalismo Técnico com Dados Visuais

**Design Movement:** Data Visualization Minimalism + Swiss Grid

**Core Principles:**
- Máxima clareza através de tipografia grande e espaço negativo
- Dados e métricas como protagonistas visuais (não decoração)
- Grid rigoroso de 12 colunas com alinhamento perfeito
- Cores limitadas: apenas verde, azul, branco e cinza

**Color Philosophy:**
- Verde-esmeralda (primário): #1B7A5E para confiança técnica
- Azul-céu (secundário): #2E5C8A para inovação
- Cinza neutro (texto): #3A3A3A para legibilidade
- Branco puro (fundo): #FFFFFF para clareza
- Raciocínio: Paleta reduzida cria profissionalismo; ausência de gradientes evoca rigor técnico

**Layout Paradigm:**
- Seções com altura exata (múltiplos de 60px)
- Conteúdo centralizado com max-width 1200px
- Imagens sempre à direita, texto à esquerda (consistência)
- Espaçamento vertical uniforme: 80px entre seções

**Signature Elements:**
1. Linhas horizontais finas (1px) em verde separando seções
2. Números grandes em azul como destaques (ex: "4 áreas de atuação")
3. Ícones monocromáticos em verde

**Interaction Philosophy:**
- Hover: mudança de cor de verde para azul (sem elevação)
- Cliques: feedback visual através de cor, não movimento
- Transições: instantâneas ou muito rápidas (100ms)

**Animation:**
- Entrada de números: contagem de 0 ao valor ao scroll
- Linhas decorativas: desenho animado ao carregar página (500ms)
- Hover em links: cor muda de verde para azul (150ms)
- Sem transições de página: navegação é instantânea

**Typography System:**
- Display: IBM Plex Sans (bold, 52-64px) para títulos
- Heading: IBM Plex Sans (semibold, 28px) para subtítulos
- Body: IBM Plex Sans (regular, 16px) para conteúdo
- Rationale: Fonte única (IBM Plex) cria coerência técnica; sans-serif é moderna e clara

## <response>

### Abordagem 3: Institucionalismo Elegante com Profundidade Sutil

**Design Movement:** Corporate Elegance + Depth Layering

**Core Principles:**
- Múltiplas camadas visuais (sombras, elevação, profundidade)
- Tipografia clássica com peso estratégico
- Padrões texturados sutis para evitar frieza
- Navegação persistente e clara em todos os contextos

**Color Philosophy:**
- Verde-floresta (primário): #1F4620 para autoridade técnica
- Azul-profundo (secundário): #1A3A52 para confiança
- Bege quente (fundo): #FAFAF7 para humanidade
- Cinza-carvão (texto): #2C2C2C para contraste
- Raciocínio: Cores mais escuras criam sofisticação; bege quente evita branco puro; combinação transmite expertise e acessibilidade

**Layout Paradigm:**
- Sidebar navegação fixa à esquerda (200px) em desktop
- Conteúdo principal com max-width 900px
- Cards com elevação (sombra 4px, blur 12px)
- Seções com fundo alternado: branco e bege

**Signature Elements:**
1. Barra lateral com ícones + texto (navegação persistente)
2. Cards com canto arredondado (12px) e sombra suave
3. Padrão de fundo: grade muito sutil (1px, 20% opacidade)

**Interaction Philosophy:**
- Hover em cards: elevação aumenta (8px), cor de fundo muda para azul claro
- Links: sublinhado em verde que aparece ao hover
- Navegação: item ativo tem fundo azul e texto branco

**Animation:**
- Entrada de cards: fade-in com elevação crescente (300ms)
- Hover em cards: elevação suave (100ms)
- Transição entre páginas: fade-out rápido (100ms) + fade-in (200ms)
- Ícones: mudança de cor ao hover (150ms)

**Typography System:**
- Display: Crimson Text (bold, 48px) para títulos principais
- Heading: Lora (semibold, 26px) para subtítulos
- Body: Source Sans Pro (regular, 16px) para conteúdo
- Rationale: Crimson Text é elegante e clássica; Lora é humanista; Source Sans Pro é legível e profissional

## Decisão Final

**Abordagem Selecionada: Modernismo Sustentável com Geometria Orgânica (Abordagem 1)**

**Justificativa:**
- Melhor alinhamento com identidade de consultoria ambiental e sustentável
- Formas orgânicas refletem foco em natureza, água e florestas
- Assimetria intencional cria dinamismo sem perder profissionalismo
- Tipografia mista (Playfair + Lora + Inter) cria hierarquia clara e elegante
- Padrões topográficos sutis reforçam expertise em geotecnologia
- Animações fluidas evocam fluxo de água e crescimento natural

**Cores Finais:**
- Verde profundo: #2D5016
- Azul-água: #0E7C86
- Neutro quente: #F8F9F6
- Cinza texto: #3A3A3A

**Tipografia:**
- Display: Playfair Display (títulos principais)
- Heading: Lora (subtítulos)
- Body: Inter (conteúdo)
